# Jigar Kanakhara — Portfolio

Personal portfolio for Jigar Kanakhara, an AI Engineer and MS Computer Science student at Boston University. Built with Next.js, React, TypeScript, and Tailwind CSS.

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```

## Deploy

Deployed on Vercel. Framework preset: Next.js. Root directory: `./`.
